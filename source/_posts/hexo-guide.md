## 简介

Hexo 是一个基于 Node.js 的静态博客框架，具有以下特点：

- 快速生成静态文件
- 支持 Markdown 语法
- 丰富的插件生态
- 多种主题可选

## 安装步骤

### 1. 安装 Node.js

首先需要在系统中安装 Node.js 环境。

### 2. 安装 Hexo CLI

```bash
npm install -g hexo-cli
```

### 3. 创建博客

```bash
hexo init myblog
cd myblog
npm install
```

### 4. 启动服务器

```bash
hexo server
```

## 常用命令

- `hexo new "标题"` - 创建新文章
- `hexo generate` - 生成静态文件
- `hexo server` - 启动本地服务器
- `hexo deploy` - 部署到远程服务器

## 总结

Hexo 是一个非常优秀的静态博客框架，适合技术人员使用。通过简单的命令就能快速搭建和管理自己的博客。
